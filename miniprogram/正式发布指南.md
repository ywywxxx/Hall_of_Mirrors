# 正式版本发布指南

## 📋 前提条件

1. **已注册微信小程序账号**
   - 访问 [微信公众平台](https://mp.weixin.qq.com/)
   - 完成小程序注册
   - 获取正式的 AppID

2. **完成小程序信息设置**
   - 小程序名称
   - 小程序头像
   - 小程序简介
   - 服务类目（选择"游戏" → "益智游戏"）

## 🔧 配置正式 AppID

### 方法 1：在 project.config.json 中配置

编辑 `miniprogram/project.config.json`，将 `appid` 改为你的正式 AppID：

```json
{
  "appid": "你的正式AppID",
  "projectname": "镜子迷宫"
}
```

### 方法 2：在微信开发者工具中配置

1. 打开微信开发者工具
2. 点击右上角的"详情"按钮
3. 在"本地设置"中找到"AppID"
4. 选择"使用正式 AppID"
5. 输入你的正式 AppID
6. 保存并重新编译

## 📤 上传代码

1. **在微信开发者工具中点击"上传"按钮**
2. **填写版本信息**：
   - 版本号：例如 `1.0.0`
   - 项目备注：例如 `初始版本 - 镜子迷宫游戏`
3. **点击"上传"**
4. 上传成功后，代码会保存到微信服务器

## ✅ 提交审核

1. **登录微信公众平台**
   - 访问 https://mp.weixin.qq.com/
   - 使用小程序管理员账号登录

2. **进入版本管理**
   - 在左侧菜单找到"版本管理"
   - 找到刚才上传的开发版本
   - 点击"提交审核"

3. **填写审核信息**：
   - **服务类目**：选择"游戏" → "益智游戏"
   - **标签**：填写"益智游戏"、"拼图"、"逻辑游戏"等
   - **功能页面**：选择首页路径 `/pages/login/login` 或 `/pages/index/index`
   - **审核备注**：可以填写游戏说明，例如：
     ```
     这是一个益智拼图游戏，玩家需要在网格中放置镜子，使光线路径匹配数字提示。
     游戏完全免费，无需登录即可使用。
     ```

4. **提交审核**
   - 检查信息无误后，点击"提交审核"
   - 等待审核（通常 1-7 个工作日）

## 🎉 发布上线

1. **审核通过后**
   - 在"版本管理"中会显示"审核通过"
   - 点击"发布"按钮

2. **确认发布**
   - 确认后，小程序即可正式上线
   - 用户可以在微信中搜索使用

## ⚠️ 常见问题

### Q: 为什么只能上传测试版本？

**可能原因：**
1. AppID 配置不正确
2. 小程序账号未完成认证
3. 账号权限不足

**解决方法：**
1. 确认 AppID 是否正确填写
2. 在微信公众平台检查账号状态
3. 确认是否已完成小程序认证（个人/企业）

### Q: 上传时提示"项目已存在"？

**解决方法：**
1. 在微信开发者工具中，点击"详情" → "本地设置"
2. 取消勾选"不校验合法域名"
3. 或者删除 `.project` 文件夹后重新导入项目

### Q: 如何确认 AppID 是否正确？

1. 登录微信公众平台
2. 进入"设置" → "基本设置"
3. 查看"AppID(小程序ID)"
4. 确保与 `project.config.json` 中的一致

### Q: 审核被拒怎么办？

1. 查看审核反馈，了解被拒原因
2. 根据反馈修改后重新提交
3. 常见被拒原因：
   - 服务类目选择不当
   - 缺少必要的功能说明
   - 存在违规内容
   - 小程序信息不完整
