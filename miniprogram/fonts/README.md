# 字体文件说明

## 手写奶酪体字体

如果需要使用"手写奶酪体"字体，请将字体文件放在此目录下：

1. **字体文件格式**：支持 `.ttf`、`.woff`、`.woff2` 格式
2. **推荐文件名**：`cheese-handwriting.ttf` 或 `cheese.ttf`

## 如何获取字体文件

1. 从字体网站下载（如：字客网、字体天下等）
2. 确保字体文件有使用授权
3. 将字体文件复制到此目录

## 当前配置

如果字体文件不存在，系统会使用备选字体：
- Kalam（手写风格）
- PingFang SC（苹果系统）
- Hiragino Sans GB（苹果系统）
- Microsoft YaHei（Windows系统）

这些字体都具有手写风格，可以提供类似的效果。
