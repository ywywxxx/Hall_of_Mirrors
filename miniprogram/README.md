# 镜子迷宫 - 微信小程序版

这是镜子迷宫游戏的微信小程序版本。

## 如何使用

1. **在微信开发者工具中打开项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择 `miniprogram` 目录
   - 填写你的 AppID（如果没有，可以选择"测试号"）

2. **配置 AppID**
   - 编辑 `project.config.json` 文件
   - 将 `"appid": "your-appid"` 替换为你的小程序 AppID
   - 如果没有 AppID，可以在微信公众平台注册：https://mp.weixin.qq.com

3. **编译运行**
   - 在微信开发者工具中点击"编译"
   - 预览效果

## 项目结构

```
miniprogram/
├── app.json          # 小程序配置
├── app.ts            # 小程序入口
├── pages/            # 页面目录
│   ├── index/        # 首页（选择难度）
│   └── game/         # 游戏页面
├── engine/           # 核心游戏逻辑（与 Web 版共享）
│   ├── types.ts
│   ├── trace.ts
│   ├── generate.ts
│   ├── mirrorCount.ts
│   └── calculateClues.ts
└── project.config.json  # 项目配置
```

## 功能说明

- **选择难度**：在首页选择 2×2 到 11×11 的网格大小
- **放置镜子**：点击格子循环切换镜子（空 → \ → / → 空）
- **标记无镜子**：长按格子标记为"无镜子"（灰色叉）
- **显示答案**：点击"显示答案"按钮查看正确答案
- **实时反馈**：红色数字显示当前结果与答案的差异
- **通关检测**：所有数字匹配时显示祝贺信息

## 注意事项

1. 小程序需要配置 AppID 才能正式发布
2. 首次使用需要在微信公众平台注册小程序账号
3. 开发版本可以在微信开发者工具中直接预览
4. 发布前需要提交审核

## 发布流程

1. 在微信开发者工具中点击"上传"
2. 登录微信公众平台（https://mp.weixin.qq.com）
3. 在"版本管理"中提交审核
4. 审核通过后，点击"发布"即可上线
